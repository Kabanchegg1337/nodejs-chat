# Node.js chat


## Setup
Download [Node.js](https://nodejs.org/en/download/).
Run this followed commands:

## Server
``` bash
# Go to server folder
cd server

# Install dependencies (only the first time)
npm install

# Run the local server at localhost:3001
node index

```


## App
``` bash
# Go to src folder
cd src

# Install dependencies (only the first time)
npm install

# Run the local server at localhost:3000
npm run dev

```
